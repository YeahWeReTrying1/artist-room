# Bobkov Portfolio

Динамический сайт-портфолио для графического дизайнера с админкой.

## Запуск

1. Скопируйте `.env.example` в `.env.local`
2. Задайте `ADMIN_TOKEN`
3. Установите зависимости: `npm install`
4. Запустите: `npm run dev`

## Где лежат файлы

- Все загружаемые файлы сохраняются в `public/uploads`
- Данные проектов: `data/projects.json`
- Контент страницы about: `data/about.json`

## Что можно делать в админке

- Создавать проекты по тегам:
  - айдентика
  - иконки
  - плакаты
  - логотипы
  - шрифты
  - эксперименты
  - зины и книги
- Загружать превью и медиа (фото/gif/video)
- Публиковать/снимать с публикации
- Убирать в архив и возвращать
- Редактировать страницу About (текст и фото)

## Логика отображения

- На главной есть лента `Flow` (все проекты вперемешку)
- Есть фильтр по тегам в верхнем меню
- Сортировка всегда от нового к старому по дате создания
- Детальная страница проекта только для `айдентика` и `шрифты`
